### Usage

#### Compile:

1. Open terminal in project root folder
2. Type & run make command
  ```console
  $ make 
  ``` 

#### Run:

1. After compiling go to project's "bin" folder
2. Run the project by typing in terminal ./project.o

  ```console
  $ ./project.o 
  ```
